Game Reviews API

By Teyah Brennen-Davies

ABOUT


This API forms part of my solo project created during the Northcoders software development bootcamp. It is the backend for my "Game Reviews" website, where users can read reviews of board games, vote on reviews, leave comments and more.


This is a REST API built using Node.js, Express and a PSQL database. For more details on the development of this API, see the Github repo. The review site itself was built using React - see the frontend repo for more details and check out the deployed React site [here](https://nc-board-game-reviews.netlify.app/).
