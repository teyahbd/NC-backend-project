Game Reviews API

By Teyah Brennen-Davies

ABOUT

This API forms part of my solo project created during the Northcoders software development bootcamp. It is the backend for my "Game Reviews" website, where users can post reviews of board games, vote on reviews, leave comments and more.

This is a REST API built using Node.js, Express and a PSQL database. For more details on the development of this API, see the Github repo hosted here. The frontend of this project is hosted here at the time of writing.

You can find the docs containing all possible endpoints here at /docs and the api is hosted at /api.
