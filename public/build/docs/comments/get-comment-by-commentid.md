# GET /api/comments/:comment_id

Returns the requested comment.

## Example Response

```
{
        "comment": {
          "comment_id": 59,
          "body": "I love this game!",
          "review_id": 1,
          "author": "jessjelly",
          "votes": 3,
          "created_at": "2021-03-27T19:48:58.110Z"
        }
      }
    }
```
